1. запустить сборку OpenServer
2. открыть PhpMyAdmin и создать БД с названием travel
2. загрузить проект в папку localhost (где лежит OS)
3. открыть сайт по ссылке localhost/travelsite/pages/createdb.php (проверить в functions.php настройки для соединения с БД)
4. в таблице БД roles добавить роли 1=admin, 2=customer
5. всё, сайтом можно пользоваться по ссылке localhost/travelsite

возможности: регистрация пользователей, просмотр отелей и добавление комментариев, админу доступно добавление стран, городов, отелей, фотографий, а также открывается дополнительная стр для изменения фото администраторов
